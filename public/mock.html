그래프로 보면 바로 보입니다

위 그래프에 표시된 두 개의 병목, 해소하면 연간 400만원이 넘습니다

B-01 · 승인 단일점

대표님 결재를 기다리는 2일

계약 건마다 대표 승인이 필요합니다. 승인 기준은 정해져 있지만 자동화가 안 됩니다. 담당자는 매번 전화하고 기다립니다.

📍 월 40건 × 2일 대기 = 월 40시간 손실 (240만원)
✓ 처방: 임계값(300만원) 이하 자동승인 + 예외만 알림 → 2일 → 10분
B-02 · 수기 대사

월말마다 반나절 엑셀 대사

주문대장(엑셀)과 납품확인서(PDF), 세금계산서를 매달 손으로 맞춥니다. 불일치가 나오면 다시 처음부터입니다.

📍 월 8시간 × 12개월 = 연 96시간 손실 (192만원)
✓ 처방: 자동 대사 스크립트 + 불일치 항목만 알림 → 8시간 → 10분
RU-01 · 규칙 가시화

머릿속에만 있던 할인 기준

영업사원마다 적용하는 할인율이 달랐습니다. 그래프로 규칙을 명문화하고 자동 체크로 연결했습니다.

✓ 결과: 할인율 오류 0건, 신규 영업사원 온보딩 2일 → 반일
E-03 · 흐름 가시화

계약 체결까지 걸리는 시간

문의부터 계약까지 평균 8일이 걸렸습니다. 각 단계 소요 시간이 그래프에 누적되자 승인 병목이 원인임이 바로 드러났습니다.

✓ 결과: 자동 승인 적용 후 계약 완료 평균 8일 → 2일
